# コマンド一覧
## cogs.audio_manager
### /play \<words>
*このコマンドはサーバーでのみ使用できます*<br>
wordsに指定された語句を全て含む、ドライブ内の音声ファイルを再生します。<br>
該当ファイルが複数ある場合、リスト表示し、番号を入力することで選択できます。
### /replay
*このコマンドはサーバーでのみ使用できます*<br>
前回再生されていたファイルを再生します。
### /randomplay \[words]
*このコマンドはサーバーでのみ使用できます*<br>
wordsに指定された語句を全て含む、ドライブ内の音声ファイルをランダムに1つ再生します。<br>
\[words]の指定がない場合、前回と同じ検索語句が指定されます。
### /seek \<seconds> \[words]
*このコマンドはサーバーでのみ使用できます*<br>
基本的な挙動は/playと同じですが、\<seconds>に指定した秒数から再生が開始されます。<br>
\[words]の指定がない場合、現在再生中のファイルに対しての操作になります。
### /pause
*このコマンドはサーバーでのみ使用できます*<br>
音声ファイルの再生を一時停止します。
### /resume
*このコマンドはサーバーでのみ使用できます*<br>
音声ファイルの一時停止を解除します。
### /stop
*このコマンドはサーバーでのみ使用できます*<br>
音声ファイルの再生を停止します。
### /search \<words>
*このコマンドはサーバーでのみ使用できます*<br>
基本的な挙動は/playと同じですが、検索機能のみで、再生されません。
### /volume \<0.0~1.0>
*このコマンドはサーバーでのみ使用できます*<br>
現在の音声ファイルの音量を0.0~1.0の範囲で指定できます。
## cogs.help_command
### /help
コマンドを一覧表示します。
### /alias
各コマンドのエイリアスを一覧表示します。
## cogs.manage_bot
### /check_prefix
*このコマンドはサーバーでのみ使用できます*<br>
現在のサーバーでのプレフィックスを表示します。
### /set_prefix
*このコマンドはサーバーでのみ使用できます*<br>
現在のサーバーでのプレフィックスを設定します。
### /reload
**このコマンドは開発者のみが実行できます**<br>
cogs直下の.pyファイルを再読み込みします。
### /close
**このコマンドは開発者のみが実行できます**<br>
Botを終了させます。
## cogs.others
### /neko
にゃーん
### /guild_name
*このコマンドはサーバーでのみ使用できます*<br>
現在のサーバー名を取得します。
### /changelog \[lines]
changelog.txtの内容を指定されたバージョン数分、表示します。<br>
\[lines]に数値ではなくallを指定すると全て表示します。<br>
何も指定しない場合は最新バージョンのログのみが表示されます。
### /log
**このコマンドは開発者のみが実行できます**<br>
プログラムの実行ログが保存されたファイルを送信します。
### /delete <limit>
**このコマンドは"メッセージを管理"の権限をもつ人のみが実行できます**<br>
*このコマンドはサーバーでのみ使用できます*<br>
\<limit>に指定された件数分、メッセージを連続削除します。
## cogs.voice_client
### /connect
*このコマンドはサーバーでのみ使用できます*<br>
使用者が接続しているボイスチャンネルに接続します。
### /disconnect
*このコマンドはサーバーでのみ使用できます*<br>
現在接続しているボイスチャンネルから切断します
## cogs.yojirei
### /yojirei \<word>
\<word>に指定した語句の用字例を検索します。<br>
※データ量が少なく、かつ改訂前のものであることに注意。
